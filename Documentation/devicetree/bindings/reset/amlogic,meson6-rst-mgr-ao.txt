Amlogic MesonX Reset Manager (AO domain)

Required properties:
- compatible : "amlogic,meson6-rst-mgr-ao"
- reg : Should contain 1 register ranges(address and length)
- #reset-cells: 1

Example:

	reset: reset@c8100040 {
		#reset-cells = <1>;
		compatible = "amlogic,meson6-rst-mgr-ao";
		reg = <0xc8100040 0x4>;
	};
